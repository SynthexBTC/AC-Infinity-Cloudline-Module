# AC Infinity Cloudline Module
A small card that allows you to connect an "AC Infinity Cloudline" Duct Fan directly to the Antminer Control Board.

- Dynamically regulate speed
- Avoid 100% at startup
- Simulation of the 4 original fans, with the RPMs that the control board wants to see
- Set a minimum speed if the fan is cooling the ASIC's power supply
- USB-A connector (or USB-C option) for using any cable
- Open-source firmware with Home Assistant ESPHome YAML configuration file !
